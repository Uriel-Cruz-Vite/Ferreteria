# Ferretería GUICHO Website

This repository contains the source code for the Ferretería GUICHO website, a static website built with HTML, CSS, and JavaScript.

## Project Overview

The website serves as a digital storefront for Ferretería GUICHO, a hardware store located in Tianguistengo, Hidalgo. It provides information about the store's products, services, and events, as well as contact information and links to its social media pages.

## File Structure

The repository is organized as follows:

- `css/`: Contains the stylesheet for the website (`styles.css`).
- `img/`: Contains images used throughout the website.
- `js/`: Contains the JavaScript code for the website (`script.js`).
- `*.html`: The HTML files for the different pages of the website.

## Getting Started

To run the website locally, simply open the `index.html` file in your web browser.
